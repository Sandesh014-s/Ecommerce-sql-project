# 🛒 E-Commerce Database Management System

A fully functional relational database project built using **MS SQL Server (T-SQL)** 
simulating the backend of a real-world e-commerce platform.

## 📌 Features
- 7 normalized tables (3NF): Categories, Products, Customers, Orders, Order Items, Payments, Reviews
- 3 Views for reporting and analytics
- 2 Stored Procedures with input/output parameters
- 2 Triggers for data integrity and automation
- 15 Business Queries using JOINs, CTEs, Window Functions, Subqueries

## 🛠️ Tech Stack
- MS SQL Server 2019+
- T-SQL
- SQL Server Management Studio (SSMS)

## 📂 Files
| File | Description |
|------|-------------|
| `ecommerce_mssql.sql` | Full project SQL script |
| `ecommerce_project_documentation.docx` | Project documentation |

## 🚀 How to Run
1. Install MS SQL Server + SSMS
2. Open `ecommerce_mssql.sql` in SSMS
3. Press F5 to execute the full script
4. Explore queries in Section 8

## 📊 SQL Concepts Covered
- DDL & DML (CREATE, INSERT, UPDATE, DELETE)
- Primary Keys, Foreign Keys, CHECK Constraints
- INNER JOIN, LEFT JOIN across multiple tables
- Aggregate Functions (SUM, AVG, COUNT)
- Window Functions (RANK, LAG, Running Total)
- CTEs (Common Table Expressions)
- Stored Procedures & Triggers
- Views & Indexes
